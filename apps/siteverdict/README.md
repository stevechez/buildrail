# BuildRail SiteVerdict

## Purpose

SiteVerdict is a website analysis and lead generation tool.

## Problem

Many contractor websites have problems with:

- Conversion
- Mobile experience
- SEO
- Messaging
- Trust signals

## Core Features

Future capabilities include:

- AI website audits
- Performance analysis
- Lead capture
- Improvement recommendations
- Contractor scoring

## Role in BuildRail

SiteVerdict acts as a customer acquisition tool.

It identifies opportunities and introduces contractors to BuildRail solutions.

## Future Direction

Become the diagnostic engine that helps contractors improve their digital presence.
